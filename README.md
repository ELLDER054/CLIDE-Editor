# CLIDE-Editor
**This is a toy syntax highlighter and development workplace, do not take this seriously as a product**

CLIDE, Custom Language IDE, is a full fledged code editor with modern features including syntax highlighting, auto-indentation, and files.
It even has a variety of color modes!
